# CarPang - 신차 장기렌트/리스 비교 플랫폼

국내 주요 렌터카 및 캐피탈사의 장기렌트·리스 견적을 한 번에 비교하고, 즉시출고 가능한 차량까지 빠르게 확인할 수 있는 플랫폼입니다.

## 주요 기능

- 🚀 빠른 상담 신청 폼
- 💰 즉시출고 & 최저가 프로모션
- ⭐ 실제 고객 출고 후기
- 🤝 20개 이상 제휴 파트너사
- 📱 반응형 디자인 (모바일/데스크톱)

## 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Form**: React Hook Form + Zod
- **Animation**: Framer Motion
- **Database**: Supabase
- **Deployment**: Vercel

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 환경 변수

`.env.local` 파일을 생성하고 다음 변수를 설정하세요:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
